Stream Selection Test – Final Blueprint (60 Questions / 60 Minutes) This document provides the finalized structure for the reduced 60-question version of the Stream Selection Test for 9th and 10th class students. The blueprint balances all seven dimensions of aptitude, personality, interest, and values while ensuring scientific validity and student-friendliness within a 60-minute duration.
